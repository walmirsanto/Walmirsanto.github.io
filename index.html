O Html

Mas o que significa HTML? Ele é a sigla do inglês Hypertext Markup Language, que pode ser traduzido como Linguagem de Marcação de Hipertexto.
A maioria das pessoas já viram ou ouviram falar sobre ele, mas não sabem o que de fato ele significa

Oque e?

Ele é o responsável por mostrar ao navegador como um conteúdo deve estar estruturado. No caso, é uma linguagem que pega o conteúdo “cru” e organiza na tela.
Sem o HTML, a web seria uma espécie de TXT básico, sem a presença de nenhum link sequer.
Se a página tem imagens, é ele que especifica onde e como elas serão apresentadas.
E mais: ele vai além daquilo que se vê. Se a página contém metadados para outras ferramentas, é o HTML que cuida disso também. Cada página que existe na Internet é um documento em HTML ou um Documento Dinâmico que gera um HTML para o navegador.
A partir do que está escrito no documento, o navegador irá agregar outros elementos na página, como:

  • Imagens;
  • Sons;
  • videos;
O HTML também é uma linguagem e você utiliza regras para organizar aquilo que quer dizer e o que a página irá mostrar.
Essas regras e elementos semânticos do HTML são chamados de Tags.
As Tags são usadas para formar qualquer objetivo identificável, como:
  1. Listas
  2. Paragrafos
  3. Campos de formulários
  4. videos
  5. Imagens
Com a atualização do Html4 para o HTML 5 a algumas que tag que se tornaram obsoletas como a tag < b >.
Em vez disso, a tag < strong > é recomendada para enfatizar o texto em negrito , pois ela indica um nível mais forte de ênfase do que a tag tag < b > . Outra tag e a < i > utilizada para deixar italico porem assim como a tag tag < b > não é mais considerado uma prática recomendada no HTML5.
