Angular

O Angular é um framework de javaScript de código aberto usado para construir aplicações para web baseados em uma única página dinâmica.
E um framework mantido pela google para construção de SPA(single page aplication) ou seja aplicações de pagina unica.
Uma SPA é basicamente uma aplicação construída em uma só página, na qual a interação e a navegação entre assessões de uma página ocorrem de maneira a qual não é necessário recarregar a página em cada uma dessas mudanças.
Para se ter uma idea praticamente todas as aplicações mantidas hoje pelo google, ela ja são construida com base no Angular como por exemplo o google drive.

Angular js

Antes de aprofundarmos mais e importante sabermos a diferença que a entre o AngularJs e Angular. O Angularjs: foi lançado em 2009 com uma ideia do Misko Hevery
O problema e que de 2009 a 2016(ano de lançamento do Angular02) as tecnologias haviam evoluido bastante , como o HTML5 ja havia lançado o CSS3 e etc, e o framework construido em 2009 antes dessas novas tecnlogias ele não conseguiu acompanha com boa performace essas atualizações, logo um novo angular com essas novas tecnologias foi criado.Então existe dois Angulars o Angularjs que e basicamente a sua primeira versão que hoje ja esta obsoleto praticamente ja não existe mais manuntenção e atualização desse framework

versões

O Angular não teria versão 1, o que faz com que muitos programadores entendam que o AngularJS foi a versão 1 do Angular. Contudo, como vimos não há nenhuma conexão entre os dois. Sendo assim, o Angular 2 foi a primeira versão do framework Angular. No mesmo ano de lançamento do Angular 2, mais precisamente no mês de Dezembro, foi anunciado o Angular 4. A partir disso, novas versões seriam lançadas em um tempo constante, sempre preservando compatibilidade com versões anteriores.
