CSS - Cascading Style Sheets

Documentos Css são utilizados para estilizar o conteudo de um documento HTML, ou seja , controla a apresentação de documentos de marcação
O HTML controla a estrutura de uma pagina , ao passo que o CSS determina a sua aparencia.
Vale lembra que essa folha de estilos pode esta no propio documento html , porem e visto como melhor pratica usarmo um documento a parti.

Como aplicar?

As regras do CSS podem ser aplicadas de três formas:

  • Inline: aplicados em um elemento html, dentro do documento
  • Incorporado: O elemento style permite aplica varias regras de uma vez no propio documento HTML
  • folha de estilos externa: Um documento com extensão .css permite aplica regras a um website inteiro de uma vez

Exemplo de uma regra CSS

Abaixo temos uma regra CSS que aplica três declarações: como mostrado Abaixo 


            

            p{ 
                text-align: center; 
                color: yellow;
                font-size: 20px; 
              }
olhando a estrutura do codigo acima temos o seguinte a unidade basica de construção , sendo composta por um seletor(alvo da regra) e de uma declaração(parametros de estilo)
A declaração por sua vez , é composta por um par propriedade: valor pegando o exemplo acima seria a propriedade text-align e o valor center;
  • Seletor: elemento p (regra aplicada a parágrafos)
  • declaração: text-align: center; color: yellow; font-size:20px
  • Propriedades: text-align, color e font-size
O exemplo acima de uma regra CSS indica que tudo oque estiver entre parenteses será centralizado por isso a um text-align: center logo também tudo oque estive entre a tag p
tera a coloração amarela por definição do color: yellow; e sua fonte será de 20px por definição do font-size:20px.

Tipos de seletores

Esse exemplo que foi dado com o p antes dos colchetes e denominado de seletor de tipo , ja que vai representa somente um determinado elemento. Porém temos também o seletor universão que e simbolizado pelo * ou seja 

                *{color:green;}
significa que todos os elementos dessa pagína serão verdes. Temos também os seletore de classes: na qual a sua função e aplica a elementos com atributos class definido
Por exemplo 
                .verde{color:green;} // aplica a cor verde para os elementos que possuem o atributo class="verde" 
                h1.verde{color:green;}
Seletor de ID: aplica estilo a elementos com atributos id definidos. 
            #corAzul{color:blue;}
